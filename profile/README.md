# Logitech Rapid Fire Macros  for PUBG

This zero-install executable runs in the background to automate mouse input for PUBG to enable rapid fire, delivering seamless Logitech rapid fire macros PUBG performance without any software imports. Perfect for users searching logitech rapid fire macros pubg, rapid fire macro pubg, macro pubg, or standalone solutions that work independently of Logitech G Hub.

## Core Features

- Hardware-level mouse click simulation for true rapid fire on semi-automatic weapons
- Automatic background operation with customizable fire rate control
- Persistent profile saving for multiple weapons and sensitivity setups
- Toggle activation with customizable hotkeys for instant on/off control

## Quick Start Guide (How to Run)

1. Download and extract archive
2. Open game and set Sensitivity 2.5, 800 DPI
3. Run the executable as Administrator
4. Use toggle key to activate/deactivate
